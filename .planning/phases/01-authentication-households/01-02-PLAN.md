<context>
@roommate-server/src/auth/auth.routes.ts
@roommate-server/src/auth/auth.service.ts
@roommate-server/src/auth/auth.controller.ts

Backend API details:
- POST /auth/register: { name, email, password } + x-session-id header
- POST /auth/login: { email, password } + x-session-id header
- GET /auth/refresh: requires x-session-id header
- GET /auth/logout: requires x-session-id header
- Access token returned in response body, refresh token in httpOnly session cookie

Reference: session ID must be generated client-side and sent with all requests
</context>
